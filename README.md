# Прогнозирование самооценки здоровья пациентов с мочекаменной болезнью

Курсовой исследовательский проект по построению моделей машинного обучения для прогнозирования субъективной самооценки здоровья пациентов с мочекаменной болезнью на основе реальных клинических данных.

## О проекте

Работа посвящена анализу факторов, влияющих на самооценку здоровья пациентов с мочекаменной болезнью, и построению статистических моделей прогнозирования.

В исследовании использованы методы линейной и полиномиальной регрессии, а также бинарной логистической регрессии. Основной целью являлось выявление наиболее значимых клинических и социально-демографических факторов, связанных с восприятием собственного состояния здоровья.

---

## Данные

Исходный материал получен в результате анкетирования **800 реальных пациентов урологического профиля**, проходивших лечение в медицинских учреждениях Воронежа и Воронежской области.

Каждое наблюдение содержит сведения о:

- социально-демографических характеристиках;
- анамнезе заболевания;
- наличии сопутствующих симптомов;
- условиях жизни;
- субъективной оценке собственного здоровья.

После предварительной обработки данных была сформирована итоговая выборка из **391 пациента с подтверждённой мочекаменной болезнью**, включающая **18 информативных признаков**.

> **Особенность проекта** — модели обучались исключительно на реальных клинических данных. Использование синтетических или искусственно сгенерированных наблюдений отсутствовало.

---

## Методология

В работе реализованы несколько моделей машинного обучения:

- **OLS (Ordinary Least Squares)** — классическая линейная регрессия;
- **Ridge Regression** — линейная регрессия с L2-регуляризацией;
- **Полиномиальная регрессия**:
  - генерация полиномиальных признаков;
  - отбор признаков с помощью ElasticNet;
  - финальная Ridge-модель;
- **Бинарная логистическая регрессия** для задачи стратификации пациентов по уровню самооценки здоровья.

Дополнительно выполнены:

- MICE-импутация пропущенных значений;
- стандартизация признаков;
- диагностика мультиколлинеарности;
- кросс-валидация;
- анализ остатков;
- ROC-анализ.

---

## Основные результаты

Получены следующие показатели качества моделей:

| Модель | Результат |
|---------|-----------:|
| Линейная регрессия (OLS/Ridge) | **R² ≈ 0.36** |
| Полиномиальная Ridge | **R² = 0.712** |
| Логистическая регрессия | **ROC AUC = 0.793** |
| Точность классификации | **Accuracy = 0.749** |

Полиномиальная модель почти **в два раза превзошла линейные спецификации**, что свидетельствует о выраженном нелинейном характере взаимосвязей между признаками.

Наиболее значимыми предикторами оказались:

- материальное положение семьи;
- наличие боли в поясничной области;
- трудовой стаж;
- квадратичный эффект возраста.

## Воспроизводимость

Исходный файл данных `Data.xlsx` не входит в репозиторий, поскольку содержит персональные данные реальных пациентов.

Для воспроизведения результатов необходимо:

```bash
python data_preprocessing.py
python model_health_linear.py
python model_health_poly.py
python model_health_logit.py
```

Во всех моделях используется фиксированное значение

```python
random_state = 42
```

что обеспечивает полную воспроизводимость вычислений.
